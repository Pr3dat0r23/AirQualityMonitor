# AirQualityMonitor

## Opis projektu
AirQualityMonitor to aplikacja monitorująca jakość powietrza w czasie rzeczywistym. Projekt umożliwia odczyt danych z czujnika jakości powietrza oraz ich wizualizację w interfejsie użytkownika.

## Funkcje
- Odczyt danych z czujników (np. PM2.5, PM10)
- Wizualizacja wyników (wykresy, kolory, alerty)
- Historia pomiarów
- Eksport danych do pliku

## Instalacja
1. Sklonuj repozytorium:
git clone https://github.com/Pr3dat0r23/AirQualityMonitor.git

2. Przejdź do folderu projektu:
cd AirQualityMonitor

3. Zainstaluj wymagane biblioteki
